## รายละเอียดโปรเจกต์ (Project Overview)
Dashboard นี้จะแสดงสามกราฟที่ไม่เหมือนกัน
* **Rating Distribution:** วิเคราะห์การกระจายตัวของคะแนนรีวิวในแต่ละระดับกลุ่มผู้ชม
* **Yearly Trends:** สำรวจความเปลี่ยนแปลงของคะแนนรีวิวตามปีที่ภาพยนตร์ออกฉาย
* **Genre Ranking:** จัดอันดับ 10 ประเภทหนังที่มีคะแนนเฉลี่ยสูงสุด
* **Data Transparency:** แสดงตารางข้อมูลดิบที่เชื่อมโยงกับตัวกราฟ
## ส่วนประกอบและฟีเจอร์สำคัญ
1.  **Summary Metrics:** แสดงจำนวนภาพยนตร์ทั้งหมดและคะแนนเฉลี่ยตามเงื่อนไขที่เลือกแบบ Real-time
2.  **Interactive Filter:** ตัวเลือก Dropdown สำหรับกรองข้อมูลตามระดับความเหมาะสมของเนื้อหา
3.  **Histogram Chart:** แสดงความหนาแน่นและช่วงคะแนนรีวิวที่พบบ่อย
4.  **Scatter Plot:** แสดงความสัมพันธ์ของคะแนนตามปีที่ฉายพร้อมระบบ Hover เพื่อดูชื่อเรื่อง
5.  **Bar Chart:** จัดอันดับประเภทหนังคะแนนสูงสุดพร้อมระบบสีแบบ Viridis ที่เปลี่ยนตามค่าเฉลี่ย
